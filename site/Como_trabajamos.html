Como Trabajamos

1. Relevamiento

Relevamiento implica recolección de datos, de las formas que sean necesarias para resolver el problema de tú empresa. Sólo con base en datos relevantes y pertinentes se puede construir una propuesta de solución que genere valor real.

2. Diagnóstico

Con los datos, se proceden a aplicar herramientas de diagnóstico específicas del área a trabajar. Esto permite catalogar los problemas, y construir las alternativas de solución. Con la presentación del problema detectado, es más fácil atacar las causas y no los síntomas.

3. Propuesta de solución

La propuesta de solución implica motivos del problema, cómo abordarlo, tiempos y recursos para la misma, así cómo resultados esperados

4. Implementación

El acompañamiento del plan de acción desarrollado, siguiendo y corrigiendo los desvíos que puedan ocurrir, para que el cambio propuesto sea perdurable.

5. control

El proceso de control se corresponde con una implementación periódica de contrastación de indicadores deseados contra obtenidos detectando de manera temprana desvíos, y aplicando las correcciones necesarias.
